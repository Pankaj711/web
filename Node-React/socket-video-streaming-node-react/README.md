Guide for local deployment -

git clone https://github.com/Pankaj711/socket-video-streaming-node-react.git

```bash
cd socket-video-streaming-node-react
```
3. Install dependencies
```bash
npm install
```
4. Change the working directory
```bash
cd client
```
5. Install dependencies
```bash
yarn install
```
6. Run the app
```bash
npm start in parent directory /socket-video-streaming-node-react
```
```bash
yarn start in directory /socket-video-streaming-node-react/client
```
7. Open project at http://localhost:3000
